# Anagrama Válido

Dado duas strings `s` e `t`, retorne `true` se `t` é um anagrama de `s`, e `false` caso contrário.

> Um Anagrama é uma palavra ou frase formada através do rearranjo de letras de uma palavra ou frase diferente, usando tipicamente todas as letras originais exatamente uma vez.

**Exemplo 1:**

```
Entrada: s = "anagram", t = "nagaram"
Saída: true
```

**Exemplo 2:**

```
Entrada: s = "rat", t = "cat"
Saída: false
```

## Limitações

- `s` e `t` são minúsculas e pertencem ao alfabeto Português.
